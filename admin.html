📋 Google Sheets Setup Instructions Click to expand
  1. Go to sheets.google.com and create a new spreadsheet named ITCA 2026 Results
  2. Click Extensions → Apps Script
  3. Delete any existing code and paste the entire contents of google-apps-script.js (the third file you downloaded)
  4. Click Save (the floppy disk icon), then click Deploy → New deployment
  5. Set type to Web app, set "Execute as" to Me, set "Who has access" to Anyone, then click Deploy
  6. Copy the Web app URL and paste it into the field above
  7. Click Save — you're connected!
